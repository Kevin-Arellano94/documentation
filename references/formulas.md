# System Formulas

## Compliance Check-in %

```
SuccessfulCheckins / TotalNumberCheckins
```

## Device Check-in %

```
(SuccessfulCheckins + FailedCheckins) / TotalNumberCheckins
```

## Failed Check-in %%

```
(1-(MissedCheckins + FailedCheckins)) / TotalNumberCheckins
```

## Successful Check-in %

```
(SuccessfulCheckins + IncompleteCheckins) / TotalNumberCheckins
```
